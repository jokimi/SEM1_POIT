HTML5 — последняя и самая мощная версия стандарта HTML. Под термином HTML5 также подразумевают набор технологий для разработки сайтов и веб-приложений. Однако, некоторые из технологий HTML5 до сих пор не утверждены окончательно. В HTML5 есть программные интерфейсы для передачи видео и звука, для чатов, в том числе видеочатов через браузер, и т. д. Например, YouTube сейчас работает через HTML5, хотя раньше обязательно требовал Flash.
В CSS каждое правило состоит из селектора и блока объявлений. Селектор определяет, на какие части документа распространяется правило. Блок объявлений помещается в "{ }" и состоит из 1 или более объявлений, разделенных ";".
